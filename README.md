## 📘 Descripción general del sistema
Sistema orientado a objetos en Java que gestiona el catálogo de tours de la agencia **Llanquihue Tour**. Lee datos desde un archivo `.txt`, instancia objetos `Tour` y los almacena en un `ArrayList`. Incluye recorrido y filtrado de la colección.

---

## 🧱 Estructura del proyecto
📁 src/

├── model/      → Clase Tour

├── data/       → Clase GestorDatos

└── ui/         → Clase Main

📁 resources/

└── tours.txt

## ⚙️ Instrucciones para ejecutar
1. Abrir el proyecto en IntelliJ IDEA
2. Verificar que `resources/` esté en la raíz del proyecto
3. Ejecutar `Main.java` desde el paquete `ui`
